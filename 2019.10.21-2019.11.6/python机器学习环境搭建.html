python机器学习环境搭建

实验

1、建议使用的工具：

  • 基本环境：python3
  • 包管理工具：anaconda
  • 在线程序编辑工具：jupyter notebook

2、安装环境：

  • 操作系统：ubuntu16.0.4LTS
  • python版本：python2.7（默认）/3.5

3、安装步骤：

（1）python安装

  • 查看python 3.5是否安装
  1. $ whereis python

如果其中存在“/usr/bin/python3.5”，则说明其中安装了python3.5。

  • 如果不存在，使用以下命令安装python3.5。
  1. $ sudo apt-get install python3.5
  • 查看ubuntu中默认的python版本
  1. $ python --version  //一般默认使用python2.7
  • 更改默认python版本
  1. $ sudo rm /user/bin/python  //删除原有的python连接文件
  2. $ ln -s /usr/bin/python3.5 /usr/bin/python  //连接python3.5文件
  3. $ PATH=/usr/bin:$PATH  //将/usr/bin加入环境变量中
  4. $ python --version  //验证默认版本，已更改为python3.5

注意：也可以下载使用更高的python版本，但是其它版本不能通过以上命令安装，需要自己下载安装包，可以参考教程。但是，不建议这么做，网上实操会出现很多问题。

（2）安装anaconda

  • 在官网上下载anaconda，建议下载最新版的【到2019.10.31为python3.7版本】。
  • 在~/download中，存在一个xxx.sh文件【相当于安装包，Anaconda3-2019.10-Linux-x86_64.sh】。
  • 校验
  1. $ sha256sum ~/download/xxx.sh  //校验该安装包是否下载完整，也可以将该安装包放在别的位置，但需要该安装包的绝对路径
  • 安装命令
  1. $ bash ~/downloads/xxx.sh  //安装
  • 阅读协议条款，并同意：“In order to continue the installation process, please review the license agreement”，持续按Enter，直到出现“Yes or no”，输入 “Yes” 。
  • 按Enter同意默认的安装位置，建议不要更换安装位置，一般为~/anaconda<2 or 3>。【ctrl-c停止安装；或者指定其它的安装路径。】
  • 将anaconda路径加入环境变量中：“Do you wish the installer to initialize Anaconda3 by running conda init?”，输入 “Yes”。
  • 安装完成，“Thank you for installing Anaconda<2 or 3>!”。可以使用 “conda --version” 验证。
  • 关闭当前窗口，再打开，发现出现 （base），说明anaconda环境被激活。可以使用 “conda config --set auto_activate_base False” 关闭anaconda环境。再启动可以使用 “conda activate” /关闭使用 “conda deactivate”。

（3）安装jupyter notebook

  • 如果anaconda成功安装，那么jupyter notebook也已成功安装。
  • 激活anaconda环境
  1. $ conda activate  //出现（base）
  • 打开jupyter notebook
  1. $ jupyter notebook

附：删除ubuntu16.0.4LST中不常用的系统软件。参考不常用的软件和彻底清除步骤。

注意：anaconda和jupyter notebook的使用方法建议参考官网。
